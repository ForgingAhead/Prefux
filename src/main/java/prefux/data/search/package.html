Text search support for prefix, keyword, and regular expression searches. Each search engine type subclasses {@link prefux.data.search.SearchTupleSet}, an implementation of the {@link prefux.data.tuple.TupleSet} interface. Classes derived from SearchTupleSet (e.g., {@link prefux.data.search.PrefixSearchTupleSet}, {@link prefux.data.search.KeywordSearchTupleSet}, and {@link prefux.data.search.RegexSearchTupleSet}) can be instantiated and used directly or in conjunction with a {@link prefux.data.query.SearchQueryBinding}.
